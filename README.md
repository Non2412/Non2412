## 📌 Featured Projects

### 📱 Smart Receipt Management System (OCR + LINE Chat Bot) | *2025 - 2026*
An automation system designed to eliminate manual receipt logging.
- **Features:** Automated receipt image collection using **LINE Chat Bot**, extraction of receipt data using **OCR** technology, and real-time dashboard display on a web application.
- **Tech Stack:** Next.js (Frontend), Python Django (Backend API), PostgreSQL (Database).
- **Impact:** Drastically reduced manual data entry time and minimized human error.

### 🛍️ Sisaket Charity (Web E-commerce Platform) | *2025*
A charity clothing web application to simplify fundraising and merchandise purchases.
- **Features:** Catalog display, product ordering, purchase data management system, and secure checkout simulation.
- **Tech Stack:** Next.js (Frontend), Backend API integration, Database management.
- **Impact:** Enhanced community engagement by connecting charity donations directly with merchandise sales.

### 🎪 Walking Street Booking System (Web Application) | *2026*
A space reservation system for merchants at walking streets to solve double-booking issues.
- **Features:** Clean UI/UX for choosing and reserving market spaces, real-time booking status (Available/Reserved), and RESTful API data sync.
- **Tech Stack:** Frontend Web Application, RESTful API, database backend.
- **Impact:** Systematized community market management and completely prevented reservation overlaps.

- **Tech Stack:** Flutter & Dart.
